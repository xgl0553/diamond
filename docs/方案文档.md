# 天天基金 2026 年度收益 Top200 基金持仓分析方案文档

## 1. 背景与目标

本方案用于使用 Python 从天天基金 / 东方财富公开页面拉取指定年度收益排名靠前的基金，获取这些基金最新披露的股票持仓，计算相对上一报告期的持仓变化，并以 CSV 数据文件与图表形式输出分析结果。

默认分析目标为：

- 年度：`2026`
- 范围：年度收益排名 Top 200 基金
- 数据源：天天基金 / 东方财富公开页面
- 输出：基金排行、基金持仓明细、股票汇总数据、PNG 图表；可选 SVG 图表

> 注意：基金持仓来自基金定期报告披露，通常存在披露滞后，不代表实时调仓结果。本方案仅用于数据分析与可视化展示，不构成投资建议。

## 2. 数据来源

### 2.1 基金收益排行数据

基金收益排行使用天天基金排行接口：

```text
https://fund.eastmoney.com/data/rankhandler.aspx
```

核心请求参数：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `op` | `ph` | 排行接口操作类型 |
| `dt` | `kf` | 开放式基金 |
| `ft` | `all` | 基金类型，支持 `gp`、`hh`、`zs`、`qdii` 等 |
| `sc` | `jn` | 按今年以来收益排序 |
| `st` | `desc` | 降序排列 |
| `sd` | `2026-01-01` | 收益统计开始日期 |
| `ed` | 当前日期与 `2026-12-31` 中较早者 | 收益统计结束日期 |
| `pn` | `200` | 返回基金数量 |

### 2.2 基金股票持仓数据

基金持仓使用基金 F10 档案接口：

```text
https://fundf10.eastmoney.com/FundArchivesDatas.aspx
```

核心请求参数：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `type` | `jjcc` | 基金持仓数据类型 |
| `code` | 基金代码 | 需要查询的基金代码 |
| `topline` | `10` | 每只基金抓取前 N 大股票持仓 |
| `year` | `2026` 和必要时 `2025` | 用于获取最新报告期和上一报告期 |

## 3. 总体流程

```text
启动脚本
  |
  |-- 解析命令行参数
  |
  |-- 拉取指定年度收益 Top N 基金排行
  |     |-- 解析接口返回的 datas 字段
  |     |-- 规范化基金代码、名称、净值、收益率等字段
  |     |-- 输出 top_funds_YYYY.csv
  |
  |-- 并发拉取每只基金的持仓明细
  |     |-- 解析基金持仓 HTML 表格
  |     |-- 识别最新报告期与上一报告期
  |     |-- 规范化股票代码、股票名称、持仓权重、持股数、市值
  |     |-- 计算持仓变化：增持 / 减持 / 新进 / 持平或小幅变动
  |     |-- 输出 fund_latest_holdings_YYYY.csv
  |
  |-- 汇总股票维度数据
  |     |-- 统计持有该股票的基金数量
  |     |-- 汇总持仓权重
  |     |-- 统计增持、减持、新进基金数量
  |     |-- 输出 stock_summary_YYYY.csv
  |
  |-- 输出图表
        |-- 基金收益 Top30
        |-- 股票持有基金数量 Top30
        |-- 股票持仓权重 Top30
        |-- 股票增减仓 / 新进统计 Top30
        |-- 基金-股票持仓权重热力图
```

## 4. 功能设计

### 4.1 命令行参数

脚本入口为：

```bash
python tiantian_fund_holdings.py --year 2026 --top 200 --out outputs
```

常用参数：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `--year` | `2026` | 收益统计年份 |
| `--top` | `200` | 拉取收益排名前 N 只基金 |
| `--fund-type` | `all` | 基金类型筛选 |
| `--holdings-topline` | `10` | 每只基金抓取前 N 大股票持仓 |
| `--max-workers` | `8` | 并发抓取持仓线程数 |
| `--out` | `outputs` | 输出目录 |
| `--chart-format` | `png` | 图表格式，支持 `png`、`svg`、`both` |
| `--timeout` | `20` | HTTP 超时时间，单位秒 |
| `--retries` | `3` | HTTP 失败重试次数 |

### 4.2 排行数据解析

排行接口返回 JavaScript 风格数据，脚本通过正则提取 `datas` 字段，并将每条基金记录拆分为结构化字段。

输出字段包括：

- 基金代码
- 基金名称
- 净值日期
- 单位净值
- 累计净值
- 日涨幅、周涨幅、月涨幅等收益字段
- 今年以来收益率
- 排名
- 排行统计起止日期

### 4.3 持仓数据解析

基金持仓接口返回包含 HTML 表格的 JavaScript 字符串。脚本处理步骤为：

1. 提取接口响应中的 `content` 字段。
2. 使用轻量 HTML 表格解析器读取表格。
3. 根据表格前文识别报告期，例如 `2026年1季度`、`2025年4季度`。
4. 从持仓表中识别股票代码、股票名称、持仓占比、持股数、市值等列。
5. 清洗百分号、千分位、缺失值等格式。

### 4.4 持仓变化计算

脚本以最新报告期为当前持仓，以紧邻的上一报告期为对比基准。

变动规则：

| 状态 | 判定规则 |
| --- | --- |
| `增持` | 当前持股数大于上一报告期持股数 |
| `减持` | 当前持股数小于上一报告期持股数 |
| `新进` | 当前报告期持有，但上一报告期未出现 |
| `持平/小幅变动` | 当前和上一报告期均出现，且持股数无明显变化或无法计算差异 |
| `无上期对比` | 未获取到上一报告期数据 |

输出字段包括：

- 当前报告期
- 上一报告期
- 当前持仓权重
- 当前持股数
- 当前持仓市值
- 上期持仓权重
- 上期持股数
- 权重变化百分点
- 持股数变化
- 变化状态

## 5. 输出设计

### 5.1 CSV 输出

输出目录默认为 `outputs`。

| 文件 | 说明 |
| --- | --- |
| `top_funds_YYYY.csv` | 指定年度收益 Top N 基金排行 |
| `fund_latest_holdings_YYYY.csv` | 每只基金最新持仓及相对上一报告期变化 |
| `stock_summary_YYYY.csv` | 按股票聚合后的持仓基金数、权重、增减仓统计 |

### 5.2 图表输出

默认输出 PNG 图表，若指定 `--chart-format svg` 则输出 SVG，指定 `--chart-format both` 则同时输出 PNG 和 SVG。

| 图表 | 说明 |
| --- | --- |
| `fund_ytd_top30.png` | 年度收益排名前 30 基金柱状图 |
| `stock_by_fund_count_top30.png` | Top 基金中被持有次数最多的股票 |
| `stock_by_weight_top30.png` | Top 基金重仓股票权重汇总 |
| `stock_change_status_top30.png` | 增持 / 减持 / 新进基金数最多的股票 |
| `fund_stock_weight_heatmap.png` | 基金-股票最新持仓权重热力图 |

## 6. 运行环境

建议环境：

- Python 3.10+
- 可访问天天基金 / 东方财富相关域名的网络环境
- Matplotlib 3.8+

安装依赖：

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

运行示例：

```bash
python tiantian_fund_holdings.py --year 2026 --top 200 --out outputs
```

小规模验证：

```bash
python tiantian_fund_holdings.py --year 2026 --top 5 --out outputs-test --timeout 20 --retries 3
```

## 8. 风险与限制

1. **数据披露滞后**：基金持仓来自定期报告，不代表实时持仓。
2. **页面结构变化风险**：天天基金 / 东方财富页面或接口结构变化可能导致解析失败。
3. **网络访问限制**：部分服务器、容器或代理环境可能无法访问目标站点。
4. **持仓口径限制**：默认只抓取前 10 大股票持仓，可通过 `--holdings-topline` 调整。
5. **权重汇总口径**：股票权重汇总未按基金规模加权，仅为 Top N 基金披露持仓权重的简单汇总。
